Daily Forex Volatility Filter using GARCH + VIX correlation.

## What It Does
- Forecasts if tomorrow will be a good trading day
- Uses GARCH(1,1) volatility model
- Considers VIX correlation
- Analyzes 7 major forex pairs

## How to Run

### Option 1: Free Online Runners
1. Go to [PythonAnywhere](https://www.pythonanywhere.com)
2. Create free account
3. Upload files
4. Run in console: `python forex_volatility.py`

### Option 2: Google Colab (Easiest)
1. Open [Google Colab](https://colab.research.google.com)
2. Create new notebook
3. Upload `forex_volatility.py`
4. Run each section

### Option 3: Replit
1. Go to [Replit](https://replit.com)
2. Create Python repl
3. Upload files
4. Click Run
